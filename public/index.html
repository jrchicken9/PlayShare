How PlayShare works

Get started in a few steps—same idea as other watch-party extensions, tuned for clarity.

  1. Install PlayShare from the Chrome Web Store (button above).
  2. Pin the extension so the PlayShare icon stays easy to reach in your toolbar.
  3. Open a video on a supported site and pick something to watch.
  4. Start a room from the extension, then share the room code or invite link with friends.
  5. Join a room on the same title—everyone keeps their own provider login; playback and chat sync through PlayShare.
