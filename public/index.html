Start in minutes

From install to first room

No second bill from us—just the extension and the streaming accounts your group already has.

  1. Add PlayShare via the install options at the top (Chrome Web Store when live, or the .zip for sideload).
  2. Pin it in Chrome so the icon stays visible on watch sites.
  3. Open a title on any supported service from the list above.
  4. Create a room from the extension popup, then drop the code or invite link to your friends.
  5. They join the same show with their own logins; PlayShare keeps playback and the sidebar aligned.

Before you go

Add PlayShare, then press play

At the top of this page, pick the Chrome Web Store (when it’s live) or the .zip download. Add PlayShare to Chrome, open something on a supported service, press play, and share a room—your friends stay on the same moment with you.
