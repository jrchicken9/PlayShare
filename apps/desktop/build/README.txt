icon.icns is generated from ../../shared/brand-mark.png for the macOS app bundle (Dock, Finder, DMG). Regenerate on a Mac after changing the brand mark by running, from the repository root, "node scripts/build-desktop-mac-icon.mjs", or "npm run icons:desktop-mac" when that script is present in the root package.json. Running "npm run dist:mac" from apps/desktop also refreshes this file before packaging (electron-builder reads build/icon.icns).
